# TOM TROC - README (Déploiement/installation)

## Prérequis

Pour faire fonctionner le projet en local :

- PHP 8.x
- Serveur web local (Apache / WAMP / XAMPP / MAMP)
- MySQL
- Navigateur moderne (Chrome / Firefox...)

## Installation du projet

Clôner ou copier le projet et placez-le dans votre dossier web local.

## Importer la base de donnée

Créer une base de donnée tomtroc et importer le fichier tomtroc.sql
Dedans se trouve un jeu de données de test.

## Configurer la connexion à la base de donnée

Si nécessaire, adaptez votre connexion dans config/databse.php (host, dbname...)

## Lancer le site

Vous pourrez accéder au site en allant sur : "http://localhost/tom_troc_web/index.php".
